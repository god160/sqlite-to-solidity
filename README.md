# SQLite → Solidity Query Generator

Free tool to generate Solidity `string.concat` queries for Tableland from SQLite statements.

## 🚀 Live Demo
https://god160.github.io/sqlite-to-solidity

## Features
- SELECT, INSERT, UPDATE, DELETE, CREATE TABLE support
- Dynamic column/value input
- WHERE clause builder
- Generate production-ready Solidity code
